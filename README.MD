# Tag Game

A multiplayer tag game built in Java. Players choose abilities and chase each other around the screen.

Built on top of three custom libraries:
- **[ScreenDisplay](https://github.com/JadenSummerville/ScreenDisplay)** — abstracts Java's Swing framework for placing and moving images on screen and handling keyboard input
- **[Ticker](https://github.com/JadenSummerville/Ticker)** — keeps the game loop running on a consistent timer regardless of processing load
- **[collision-detector](https://github.com/JadenSummerville/collision-detector)** — handles collision detection between players

## How to Run

Requires Java to be installed. From the project root:
```
run
```
Configure game settings before starting.
```
run_game
```
Skip to the game directly.
